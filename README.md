## Simple Progressive Web Application in React

### How to run:

1. Have a local sever running with GET /posts (`json-server` can be used)
2. `yarn global add serve`
3. `yarn build`
4. `serve -s build`

### Note

All the required changes are tagged with `PWA` inside the code
